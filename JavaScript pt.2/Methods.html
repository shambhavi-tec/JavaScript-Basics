• Trim Method
    • The trim() method removes whitespace from both ends of a string. (space, tab, no-break space, etc.)
  • toUpperCase Method
    • The toUpperCase() method converts a string to uppercase letters.
  • toLowerCase Method
    • The toLowerCase() method converts a string to lowercase letters.
  • NOTE: Strings are immutable
    • This means that the string itself is not changed. Methods return a new string.
